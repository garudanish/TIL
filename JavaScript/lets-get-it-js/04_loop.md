# 04. 반복문

> Let's Get IT 자바스크립트 프로그래밍

## `while` 문

```js
while (조건식) 실행문;
```

`while` 문은 조건식이 참인 동안 반복해서 실행문을 실행한다. 실행문이 하나이면 중괄호를 적지 않아도 되지만, `if`문과 마찬가지로 반복문의 범위를 명확하게 하기 위해서 실행문이 하나여도 중괄호를 적는 것이 좋다.

```js
while (true) {
  console.log("Hello, while!");
}
```

위 반복문을 실행하면 조건식이 참이므로 `console.log("Hello, while!")`이 무한히 실행된다. 실제로 실행하면 무한히 찍히다가 웹 브라우저가 뻗어버린다. 이런 일을 방지하기 위해 프로그래밍할 때는 무한 반복되는 상황을 방지해야 한다.

반복문을 멈추려면 원하는 만큼 반복됐을 때 조건식을 `false`로 만들어주면 된다.

```js
let i = 1;
while (i <= 100) {
  console.log("Hello, while!");
  i++;
}
```

위 반복문을 시행하면 `console.log`가 100번 실행된다.

변수 `i`를 선언하고 조건식을 `i <= 100`으로 준다. 그리고 `console.log`를 할 때마다 `i`를 하나씩 올린다. `i++`는 `i += 1`과 같다. `i`가 101이 되면, 즉 `console.log`를 100번 출력하면 조건식이 `false`가 되어 반복문이 멈춘다.
실제로 프로그래밍 할 때는 변수에 첫 번째 값으로 1보다는 0을 많이 넣는다. 프로그래밍에서는 숫자를 0부터 세기 때문이다. 위 코드에서 0부터 시작할 때면 조건식을 `i <= 99` 혹은 `i < 100`으로 바꿔야 한다.

## `for`문

```js
for (시작; 조건식; 종료식) {
  실행문;
}
```

```js
for (let i = 0; i < 100; i++) {
  console.log("Hello, for!");
}
```

시작 부분(`let i = 0;`)이 먼저 실행되고 나서, 조건식(`i < 100`), 실행문(`console.log("Hello, for!")`), 종료식(`i++`)의 순서대로 실행된다. 그리고 다시 조건식을 검사해 만약 조건식이 참이라면 실행문과 종료식을 반복해서 실행한다. `for`문은 `while`문에서는 흩어져 있던 반복문 요소들을 모아서 표현하고 있는 형태이다.

## 1부터 100까지 출력하기

```js
let i = 0;
while (i < 100) {
  console.log(i + 1);
  i++;
}
```

```js
for (let i = 0; i < 100; i++) {
  console.log(i + 1);
}
```

## `break`문으로 반복문 멈추기

반복문을 중간에 멈춰야 하는 경우가 있다. 자연수나 실수 전체를 대상으로 반복문을 돌리는 경우, 반복문을 돌릴 대상이 무한히 많아진다. 반복문으로 값을 하나씩 찾다가 원하는 값을 찾으면 반복문을 멈춰야 한다. 멈추지 않으면 반복문이 계속 실행되므로 다음 코드를 실행할 수 없고 찾은 값을 사용할 수 없다.
반복문을 멈추기 위해 사용하는 것이 `break`문이다.

```js
let i = 0;
while (true) {
  if (i === 5) break;
  i++;
}
console.log(i); // 5
```

`while(true)`이기 때문에 위의 반복문은 무한히 반복되어야 한다. 하지만 실행문 안에 `break`문이 들어있어서, `i === 5`가 되는 순간 반복문이 종료된다. 따라서 위의 반복문은 무한히 반복되지 않는다.

무한 반복문을 표현할 때는 `while`문이 더 간결하므로 `for`문 보다 `while`문을 더 많이 사용한다. 하지만 두 경우 모두 `break`문 없이는 의미가 없다.

## `continue` 문으로 코드 실행 건너뛰기

반복문이 특정 조건에서만 실행되기를 원할 때는 `continue` 문을 사용한다. `continue` 문을 넣으면 이후 코드는 건너뛰게 된다.

```js
let i = 0;
while (i < 10) {
  i++;
  if (i % 2 === 0) {
    continue;
  }
  console.log(i);
}

// 1
// 3
// 5
// 7
// 9
```

`i % 2 ===0`, 즉 짝수일 때는 `continue`문을 넣어서 이후 코드인 `console.log`를 건너뛰게 하고, 홀수일 때만 `console.log`를 실행시켰다.

## 중첩 반복문 사용하기

반복문 안에 반복문이 들어있는 경우를 중첩 반복문이라고 한다.

```js
for (let i = 0; i < 10; i++) {
  for (let j = 0; j < 10; j++) {
    console.log(i, j);
  }
}
```

위의 중첩 반복문을 시행하면 `0 0`, `0 1`, `0 2`, ... `9 7`, `9 8`, `9 9` 순으로 반복문이 돌아간다. `i`가 0일 때 `j`가 0부터 9까지 반복문이 실행되고, 그 다음 `i`가 1일 때 `j`가 0부터 9까지 반복문이 실행되고...의 방식이다.
